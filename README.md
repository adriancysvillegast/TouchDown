# TouchDown

This app was made using SwiftUI.
This is an E-commerce app.
topics: 
- Environment properties
- Observable Objects
- Work with custom shape (Shape path)
- Grid Layout
- Navigation bar
- Gestures with UIImpactFeedbackGenerator (haptics)
